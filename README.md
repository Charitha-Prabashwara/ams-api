AMS is an **open-source, web-based Academic Management System** designed to manage and streamline the academic activities of educational institutions using a **modern, modular, API-driven architecture**.

The system is built using **Express.js** and developed in independent components, which are later integrated with frontend modules through a **CI/CD pipeline**, enabling easy updates, maintenance, and scalability.

---

## 🎯 Project Goals

- Replace **informal academic communication systems** with a structured solution  
- Practice **software engineering discipline** as a Software Technology undergraduate  
- Apply **software design patterns** in a real-world project  
- Build a system that is **scalable, maintainable, and easy to update**  
- Follow industry practices such as **modular architecture and CI/CD**

---

## 🧠 Motivation

Many universities rely on **smart messaging platforms such as WhatsApp** for academic operations. This approach is unstructured and unreliable for managing critical academic information.

In my own university experience, dependence on such platforms directly contributed to my inability to appear for the **Data Structures and Algorithms (Theory)** examination. While this was not the fault of individual lecturers, the reliance of a state university on foreign social media platforms for core academic processes is deeply problematic.

Instead of remaining frustrated, I chose to **transform that frustration into a constructive solution**. AMS is my response—both technical and personal—to build a **centralized, professional academic management system** that reflects how academic systems *should* function.

---

## 🧩 System Design Philosophy

- **API-first architecture**
- **Express.js-based backend**
- **Modular development**
- **Loose coupling** between backend and frontend
- **CI/CD-based integration**
- **Design-pattern–oriented implementation**
- **Easy extensibility and long-term maintainability**

The system evolves incrementally rather than being developed as a monolithic application.

## 🛠️ Technology Stack
